# AI Expense Tracker (React)

A React + Vite version of the uploaded AI expense tracker dashboard.

## Features

- Reusable React component structure
- Chart.js line and doughnut charts via `react-chartjs-2`
- Mock AI advisor interaction with rotating responses
- Filterable transaction list
- Responsive dark dashboard layout

## Getting started

```bash
npm install
npm run dev
```

## Build for production

```bash
npm run build
npm run preview
```

## Project structure

- `src/App.jsx` — page composition and state
- `src/components/` — reusable dashboard sections
- `src/data/dashboardData.js` — mock data and UI content
- `src/styles.css` — dashboard styling

## Notes

This project was adapted from the uploaded HTML prototype and converted into a React app structure for local development and extension.
